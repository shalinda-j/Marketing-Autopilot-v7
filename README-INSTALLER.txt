The Windows installer for Marketing Autopilot has been successfully created!

Location of the installer:
  dist_electron/Marketing Autopilot-Setup-0.0.0.exe

To install the application:
1. Navigate to the dist_electron folder
2. Double-click on "Marketing Autopilot-Setup-0.0.0.exe"
3. Follow the installation wizard steps
4. The application will be installed on your system
